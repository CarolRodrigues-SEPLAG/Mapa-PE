# Mapa de Pernambuco por Município

Aplicativo Streamlit para gerar mapa municipal de Pernambuco com nomes fixos, upload de indicadores por município ou código IBGE, coloração por escala automática ou faixas personalizadas e download em SVG para relatório.

## Arquivo principal

Use este arquivo ao publicar no Streamlit Cloud:

```text
app_mapa.py
```

## Rodar localmente

```powershell
python -m streamlit run app_mapa.py --server.port 8502 --server.address localhost
```
